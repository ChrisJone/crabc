# crabc

#### 介绍
Crabc是低代码开发平台，企业级通用基础辅助系统，
支持（mysql、oracle、postgresql、sqlserver、es等）
将数据快速生成API接口提供服务，提供了一键生成通用CRUD接口方法，
可通过插件的方式扩展支持其他的数据源,支持接口转发、黑白名单、权限认证、限流、缓存、监控等功能
